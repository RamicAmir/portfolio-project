# Portfolio-project

## Run To config
```
docker-compose config
```
## Run To build
```
docker-compose build
```
## Run To rebuild
```
docker-compose up --build
docker-compose up -d  
```
## Run To stop 
```
docker-compose stop
```
## Run To remove
```
docker-compose rm
docker-compose down
```
