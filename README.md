# Portfolio-project

## Run to config
```
docker-compose config
```
## Run to build
```
docker-compose build
```
## Run to rebuild
```
docker-compose up --build
docker-compose up -d  
```
## Run to stop 
```
docker-compose stop
```
## Run to remove
```
docker-compose rm
docker-compose down
```
